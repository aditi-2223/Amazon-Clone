# ------ Amazon Clone --------

# Overview 
A simple front-end Amazon homepage clone built using basic web development languages.
This project focuses on recreating the look and feel of Amazon’s UI while practicing responsive web design and modern CSS styling techniques.

# Features 
Responsive Amazon-inspired homepage

Navigation bar with search section

Hero/banner section

Product category cards

Footer section

Clean and organized layout

# Tech Stack
HTML5 - for structure of website 

CSS3 - for modern responsive styling

# Learning Outcomes
Through this project, I got hands-on understanding of: 

Flexbox

CSS Grid

Positioning

Responsive design

UI cloning techniques
